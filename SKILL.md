# infographic-generator - 信息图生成器

根据主题内容，选择不同风格提示词，生成高质量信息图。当前支持 **6 种风格**，覆盖教育、科技、商务、文艺、复古、暗黑等场景。

## 能力说明

- 输入主题和核心信息点
- 选择风格模板（6 种可选）
- 输出完整提示词，用于 AI 绘图工具生成信息图
- 支持直接调用 `ai-image-generation` 技能生成图片

---

## 使用方式

### 方式一：指定风格

```
生成一张【风格名】信息图：【主题内容】
```

### 方式二：让 AI 推荐风格

```
帮我生成信息图：【主题内容】
```

AI 会根据内容自动推荐最合适的风格。

## 风格速览

### 🎨 hand-drawn-edu - 手绘教育风

暖白纸张 + 马卡龙色系 + 手绘线条 + 涂鸦装饰

**适合**：知识科普、流程说明、概念对比、学习笔记

### 🧸 cartoon - 童趣卡通风

明亮糖果色 + 可爱卡通元素 + 游戏化布局 + 表情装饰

**适合**：儿童教育、趣味科普、亲子活动、游戏化学习

### 💻 geek-tech - 极客科技风

深空黑背景 + 霓虹发光线条 + 终端美学 + 数据可视化

**适合**：技术架构图、编程教程、API流程、数据分析

### 💼 corporate - 商务专业风

商务蓝主调 + 清晰数据图表 + 企业级排版 + 专业图标

**适合**：工作汇报、商业报告、项目进度、市场分析

### 🎭 artistic-illustration - 文艺插画风

水彩质感 + 手绘插画 + 莫兰迪色系 + 手账拼贴感

**适合**：旅行攻略、美食记录、生活方式、书籍推荐

### 🦇 dark-gothic - 暗黑哥特风

深黑背景 + 血红强调 + 哥特装饰 + 戏剧光影

**适合**：恐怖故事、悬疑推理、暗黑题材、万圣节主题

### 📰 vintage-newspaper - 复古报纸风

泛黄纸张 + 报刊排版 + 木刻版画 + 年代质感

**适合**：历史事件、新闻摘要、档案资料、怀旧主题

## 工作流程

1. **理解内容**：分析主题和核心信息点
2. **推荐风格**：根据内容类型推荐最合适的风格（用户也可指定）
3. **生成提示词**：读取对应风格模板，生成完整绘图提示词
4. **生成图片**：调用 `ai-image-generation` 技能或输出提示词供用户使用
