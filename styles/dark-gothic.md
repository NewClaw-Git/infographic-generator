# 暗黑哥特风格 (Dark Gothic Style)

适用于：恐怖故事、悬疑推理、暗黑题材、万圣节主题、哥特文化

---

## 完整提示词模板

```
你是一位擅长暗黑哥特风信息图的视觉设计师。请根据以下内容创作一张单页信息图。

## 风格要求

整体风格：Dark gothic infographic with mysterious atmosphere and dramatic lighting。深沉色调 + 复古装饰元素 + 戏剧性光影对比。像哥特小说插图或暗黑游戏的概念设定图。

配色方案：
- 主背景：深渊黑 #0A0A0A 或 深紫黑 #1A0A1F
- 次级背景：暗灰 #1F1F1F 或 深褐 #2D1F1A
- 主强调色：血红 #8B0000 或 深紫 #4A0E4E，用于标题、关键元素
- 次强调色：骨白 #E8E4D9（骨骼/幽灵）、暗金 #D4AF37（装饰边框）、锈红 #8B4513（血迹/锈迹）
- 渐变：紫黑渐变 #2D1F3D → #0A0A0A
- 文字：骨白 #E8E4D9，次要文字用暗灰 #696969

图形优先：用剪影、阴影、负空间承载信息。图形风格：哥特式尖拱（Gothic arches）、玫瑰窗纹样（rose window patterns）、荆棘藤蔓（thorny vines）、骷髅骨骼（skulls and bones）、蝙蝠蜘蛛等暗黑生物。

信息结构：用墓碑形状卡片、古老书页、破碎镜框等容器组织内容。连接元素：荆棘藤蔓、锁链、蜘蛛网、血迹滴落。布局不对称，带神秘感。

文字层次：标题用哥特式字体（Gothic/Blackletter font），大号锐利；内容用衬线体（serif），古典风格。文字可做旧处理（distressed text effect），像古老印刷品。

渲染细节：元素带阴影层叠（layered shadows），边缘有做旧磨损（distressed edges），角落有蜘蛛网、灰尘、裂纹装饰。整体氛围神秘、阴郁、有故事感。

底部金句：图片底部用古老书页或墓碑风格框住一句暗黑金句。

## 要图解的内容

[在这里填入主题和核心信息点]
```

## 关键视觉特征

| 元素 | 规范 |
|------|------|
| 背景色 | 深渊黑 #0A0A0A 或 深紫黑 #1A0A1F |
| 卡片色 | 暗灰 #1F1F1F 或 深褐 #2D1F1A |
| 强调色 | 血红 #8B0000 或 深紫 #4A0E4E |
| 次强调色 | 骨白 #E8E4D9、暗金 #D4AF37、锈红 #8B4513 |
| 线条风格 | 锐利、带阴影层叠 |
| 图形风格 | 哥特式尖拱 + 剪影 + 装饰纹样 |
| 装饰 | 荆棘藤蔓、蜘蛛网、骨骼、蝙蝠 |
| 布局 | 不对称、墓碑/书页容器 |
