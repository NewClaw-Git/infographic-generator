# 文艺插画风 (Artistic Illustration Style)

适用于：文艺内容、生活方式、旅行攻略、美食探店、人文故事

---

## 完整提示词模板

```
你是一位擅长文艺插画风信息图的视觉设计师。请根据以下内容创作一张单页信息图。

## 风格要求

整体风格：Soft artistic infographic with watercolor textures and hand-crafted feel。水彩质感 + 手绘插画 + 优雅排版。像旅行手账、美食杂志的文艺内页。

配色方案：
- 主背景：米白纸张 #FBF8F4 或 淡米 #F7F3EB
- 信息区色块：莫兰迪色系（Morandi colors）——灰粉 #E8D5D5、灰蓝 #B8C5D6、灰绿 #C5D5C5、奶咖 #D5C5B8
- 强调色：复古红 #B55A5A 或 藏蓝 #4A5568，用于标题、重点标注
- 渐变：暖色渐变 #F5E6D3 → #E8D5D5
- 文字：深褐 #3D3D3D，手写感

图形优先：用水彩插画、手绘小图、贴纸元素承载信息。人物用简约插画风（minimalist illustration），建筑美食用写意速写（quick sketch style）。元素带纸张撕裂边缘（torn paper edges）、胶带粘贴效果（washi tape）。

信息结构：用手账风格组织内容——便签纸（sticky notes）、拍立得照片框（polaroid frames）、贴纸装饰（stickers）、手写箭头和连线。分区用虚线框或波浪线分隔，角落有手绘小物装饰。

文字层次：标题用手写体（handwritten font），优雅流畅；正文用衬线体（serif font），文艺气质。文字可配手绘下划线、波浪线、小星星装饰。

渲染细节：元素带水彩晕染边缘（watercolor bleed edges），色块有纸张纹理叠加（paper texture overlay），整体有手工拼贴感（collage feel）。留白充足，呼吸感强。

底部金句：图片底部用手写便签或拍立得风格框住一句文艺金句。

## 要图解的内容

[在这里填入主题和核心信息点]
```

## 关键视觉特征

| 元素 | 规范 |
|------|------|
| 背景色 | 米白 #FBF8F4 或 淡米 #F7F3EB |
| 卡片色 | 莫兰迪色系（灰粉/灰蓝/灰绿/奶咖） |
| 强调色 | 复古红 #B55A5A 或 藏蓝 #4A5568 |
| 线条风格 | 手绘、柔和 |
| 图形风格 | 水彩插画 + 手绘速写 |
| 装饰 | 贴纸、胶带、便签、拍立得框 |
| 布局 | 手账风格、拼贴感 |

---

## 适用场景

- ✅ 旅行攻略（如：京都三日游）
- ✅ 美食探店记录
- ✅ 生活方式分享
- ✅ 书籍/电影推荐
- ✅ 人文故事图解
