# 极客科技风格 (Geek Tech Style)

适用于：技术架构图、编程教程、数据分析报告、产品技术文档

---

## 完整提示词模板

```
你是一位擅长科技风信息图的视觉设计师。请根据以下内容创作一张单页信息图。

## 风格要求

整体风格：Dark mode tech infographic with neon accents and terminal aesthetic。深邃背景 + 霓虹线条 + 代码元素点缀。像科技博客的技术架构图或黑客电影里的数据可视化界面。

配色方案：
- 主背景：深空黑 #0D1117 或 深蓝灰 #161B22
- 次级背景：深灰卡片 #21262D
- 主强调色：电光蓝 #58A6FF，用于标题、连接线、关键数据
- 次强调色：霓虹绿 #3FB950（成功/正向）、警告橙 #D29922（注意）、错误红 #F85149（警告）
- 渐变色：紫蓝渐变 #A371F7 → #58A6FF
- 文字：白色 #FFFFFF，次要文字用 #8B949E

图形优先：用几何图形、线条图、数据图表承载信息。节点用圆角方块或六边形（hexagons），连接线带箭头和流动动画暗示。图标风格：线性图标（line icons）或填充图标（filled icons），霓虹发光效果。

信息结构：用流程图（flowchart）、架构图（architecture diagram）、时间线（timeline）组织内容。代码块用终端风格（terminal style）展示，带语法高亮色彩。数据用进度条、环形图、柱状图呈现。

文字层次：标题用等宽字体大号（monospace, large）；副标题用斜体代码注释风格（italic code comment style）；内容用简洁技术术语（technical terms），配合图标减少文字量。

渲染细节：元素带霓虹发光边框（neon glow borders），卡片有玻璃拟态效果（glassmorphism），背景可有网格线（grid lines）、代码片段装饰（scattered code snippets）、电路图案（circuit patterns）。

底部金句：图片底部用代码注释风格输出一句总结 —— // 或 # 开头的金句。

## 要图解的内容

[在这里填入主题和核心信息点]
```

## 关键视觉特征

| 元素 | 规范 |
|------|------|
| 背景色 | 深空黑 #0D1117 或 深蓝灰 #161B22 |
| 卡片色 | 深灰 #21262D + 玻璃拟态 |
| 强调色 | 电光蓝 #58A6FF |
| 次强调色 | 霓虹绿 #3FB950、警告橙 #D29922、错误红 #F85149 |
| 线条风格 | 霓虹发光线条 |
| 图形风格 | 几何图形 + 节点连线 + 数据图表 |
| 装饰 | 网格线、代码片段、电路图案 |
| 布局 | 流程图、架构图、时间线 |

## 适用场景

- ✅ 技术架构图（如：微服务架构、前后端分离）
