# 复古报纸风 (Vintage Newspaper Style)

适用于：历史事件、新闻摘要、年代故事、档案资料、怀旧主题

---

## 完整提示词模板

```
你是一位擅长复古风信息图的视觉设计师。请根据以下内容创作一张单页信息图。

## 风格要求

整体风格：Vintage newspaper infographic with aged paper texture and classic typography。泛黄纸张 + 报刊排版 + 复古装饰。像老报纸的剪报或历史档案馆的资料卡。

配色方案：
- 主背景：泛黄旧纸 #F4ECD8 或 深米 #E8DFC8
- 次级背景：浅褐 #D5C5A8
- 主文字：深褐墨水 #2C1810 或 黑墨 #1A1A1A
- 强调色：复古红 #8B2500 或 深蓝墨 #1E3A5F，用于标题、重要标注
- 装饰色：金棕 #8B6914（边框装饰）、锈红 #8B4513（印章效果）
- 文字：深墨色 #2C1810

图形优先：用老式插图、木刻版画风格（woodcut style）、老照片框承载信息。图标风格：复古手绘图标、印章风格标记（stamp marks）。元素带纸张折痕、污渍、撕裂边缘。

信息结构：用报纸专栏风格组织内容——多栏排版（multi-column layout）、大标题（masthead）、分栏线（column rules）。引用用手写便签或打字机风格。时间线用老式火车轨道或卷轴形式。

文字层次：标题用经典衬线体（classic serif），像老报纸的标题字；副标题用斜体衬线；正文用紧凑的报刊字体（newspaper typeface）。可用首字母放大装饰（drop cap）。

渲染细节：背景有纸张纹理（paper texture）、咖啡渍、折痕。边框用老式装饰边框（ornate borders）、角落有复古花纹。整体有年代感、历史厚重感。

底部金句：图片底部用老式报纸引用框或邮票风格框住一句历史金句。
```

## 关键视觉特征

| 元素 | 规范 |
|------|------|
| 背景色 | 泛黄旧纸 #F4ECD8 或 深米 #E8DFC8 |
| 次级背景 | 浅褐 #D5C5A8 |
| 强调色 | 复古红 #8B2500 或 深蓝墨 #1E3A5F |
| 线条风格 | 墨迹、手绘感 |
| 图形风格 | 木刻版画 + 老照片框 |
| 装饰 | 纸张折痕、咖啡渍、装饰边框 |
| 布局 | 报纸多栏排版 |
