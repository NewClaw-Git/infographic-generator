# 商务专业风 (Corporate Professional Style)

适用于：商业汇报、企业宣传、工作总结、项目汇报、市场分析

---

## 完整提示词模板

```
你是一位擅长商务风信息图的视觉设计师。请根据以下内容创作一张单页信息图。

## 风格要求

整体风格：Clean corporate infographic with professional aesthetic and data-driven design。专业配色 + 清晰数据呈现 + 企业级排版。像咨询公司报告或上市公司年报的内页。

配色方案：
- 主背景：纯白 #FFFFFF 或 浅灰 #F8F9FA
- 主色调：商务蓝 #1E3A8A 或 科技蓝 #2563EB
- 次色调：深灰 #374151、浅蓝 #DBEAFE
- 强调色：活力橙 #F97316 或 成功绿 #10B981，用于数据高亮
- 图表色：蓝色系渐变 #3B82F6 → #60A5FA → #93C5FD
- 文字：深灰 #1F2937，次要文字 #6B7280

图形优先：用专业图表、图标、信息块承载信息。图表风格：简洁的数据可视化——柱状图、饼图、折线图、环形图。图标：线性商务图标（business line icons），蓝色填充。

信息结构：用网格布局（grid layout）组织内容，模块化卡片分区。数据用大号数字 + 小字标注呈现。对比用左右分栏，流程用横向步骤条，比例用环形图。

文字层次：标题用无衬线粗体（sans-serif bold），企业级专业感；正文用无衬线体，简洁清晰。数字用等宽字体突出，配合百分比或单位。

渲染细节：卡片带轻微阴影（subtle shadow），边框锐利整洁。背景可有细微网格或点阵装饰（subtle grid pattern）。整体简洁、专业、信息密度适中。

底部金句：图片底部用粗体蓝色文字居中输出一句商业洞察总结。

## 要图解的内容

[在这里填入主题和核心信息点]
```
